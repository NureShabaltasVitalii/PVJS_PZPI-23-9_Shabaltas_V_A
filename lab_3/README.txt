Лабораторна робота 3
Тема: мобільний застосунок "Простий блог" на Kotlin.

Що реалізовано:
1. Авторизація користувачів.
2. Реєстрація нового користувача.
3. Додавання статей.
4. Редагування статей.
5. Видалення статей.
6. Категорії статей: Навчання, Новини, Особисте, Інше.
7. Додавання коментарів до статей.
8. Пошук за заголовком і текстом статті.
9. Фільтрація статей за категорією.

Тестовий користувач:
Логін: admin
Пароль: 1234

Як відкрити:
1. Відкрийте Android Studio.
2. Натисніть File > Open.
3. Виберіть папку lab3_blog_app.
4. Дочекайтесь Gradle Sync.
5. Запустіть застосунок на емуляторі або телефоні.
